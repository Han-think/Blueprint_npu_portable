# UI Kit — Blueprint Artifact

Reusable React/JSX components for composing engineering blueprints in the Blueprint NPU visual language. Pair with `colors_and_type.css` at the project root.

Components:
- `Frame` — outer navy canvas + 2px frame + 16px gutter
- `Stage` — vertical stage body (S‑IC / S‑II / S‑IVB style)
- `Tank` — domed tank rectangle (LOX blue / LH2 amber / RP‑1)
- `EngineBell` — filled triangle engine with optional row count
- `Legend` — top‑right swatch legend card
- `DimLadder` — left‑gutter dimension ladder with tick caps
- `PartTreeBlock` — ```part_tree fenced JSON viewer
- `LaneChip` — pastel UPPER chip

See `index.html` for a live demo that composes these into a mini Saturn‑V‑style stack.
